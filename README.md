# situs-aplikasi-edukasi
Absen Online Barcode, Sipaguru, Digarsip, Sihebat, dan Smartbel+E-Monitoring merupakan aplikasi digital karya Watono, S.Pd untuk mendukung transformasi sekolah. Fitur meliputi presensi barcode, administrasi guru, arsip digital, pembinaan karakter siswa, serta monitoring KBM guna meningkatkan efisiensi, disiplin, dan kualitas layanan pendidikan.
